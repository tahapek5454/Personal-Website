Projeler

Tübitak Projeleri

Görme Engelliler için Basılı Dokümanlarda Sesli Metin Arama ve Seslendirme Mimarisi

Bu çalışmada görme engelli bireylerin basılı medya (dijital olmayan medya) üzerinde sesli-interaktif olarak arama yapmalarına imkan sağlayan mobil destekli bir sistem gerçekleştirilmiştir.

Çekilen görselde YOLO V8 algoritmasıyla tespit edilen metin görselleri paragraf ve liste şekilnde etiketlenmektedir. Etiketlenen veriler Google Vesion API ile image to text işlemine tâbi tutulmaktadır. Kullanıcının aramak istediği anahtar kelime aranmakta ve isteğe göre ; Tüm metin, paragraf, cümle ya da liste olacak şekilde sesli bir şekilde kullanıcıya iletilmektedir. Sistem tamamiyle sesli komutlar üzerine çalışmaktadır.

Kişisel Projeler

ASP.NETCore-6-ECommerceAPI

Bu projenin amacı en yeni teknolojilerden yararlanarak fullstack bir E-Ticaret projesi geliştirerek ASP.NETCore ortamın proje geliştirmektir.

Projede kurumsal mimarilerde sıklıkla kullanılan Onion Architecure'dan yararlanılmıştır. API kısmı içersinde CQRS Patern ve Mediatr kütüphanesinden yararlanılmıştır. Frontend tarafıyla eşzamanlı olarak server tarafından da iletilişim sağlanması adına SignalR alt yapısı inşaa edilmiştir. Veritabanı olarak cloud ortamında çalışan Postgre Sql veritabanı tercih edilmiştir. Google Authentication, Google Storage gibi çeşitli cloud teknolojilerinden yararlanılmıştır. ORM araçlarından EntityFramework kullanılmıştır. Security tarafında JWT ve Microsoft.Identity kütüphanesinden yararlanılmıştır. Frontend için Angular frameworkünden yararlanılmıştır. Ekstra olarak da birçok güncel teknolojilerden yararlanılmıştır.

ASP.NETCore-6-ECommerceAPI projesine göz atmak için github sayfama göz atabilirsiniz.

Self Waiter

Bu projenin amacı Cafe ve Restorantlarda kare kod ile menü görüntülemenin yanı sıra sipariş verme ve siparişi takip etme sistemi kurmaktır.

Bu projede de .NETCore 6 üzerinde tıpkı E-Commerce projesindeki gibi bir API gerçekleştirilmiştir. Onion Architecure'dan yararlanılmıştır. CQRS Patern ve Mediatr kütüphanesinden yararlanılmıştır. SignalR alt yapısı inşaa edilmiştir. Dinamik dil desteği sağlanmıştır. Veritabanı olarak cloud ortamında çalışan Postgre Sql veritabanı tercih edilmiştir. Google Storage gibi çeşitli cloud teknolojilerinden yararlanılmıştır. ORM araçlarından EntityFramework kullanılmıştır. Security tarafında JWT ve Microsoft.Identity kütüphanesinden yararlanılmıştır. Frontend için ise React frameworkünden yararlanılmıştır.

Self Waiter projesine göz atmak için github sayfama göz atabilirsiniz.

Yapmış olduğum daha birçok projeye göz atmak ve kaynak kodlarını incelemek için github sayfama göz atabilirsiniz.
